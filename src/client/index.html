Survive3

Select your team:

Rules: Terrain belongs to the closest player. Inactive players will be kicked automatically.

Controls: Mouse click or touch to move. Mouse wheel to zoom in and out.

System requirements: Firefox/Chromium with WebGL support. Geforce 8400 or faster. Celeron J1800 or faster.

Survivor3 is a little experiment using HTML5, jQuery, pixiJS, Java and Spring. The source code is available here.
